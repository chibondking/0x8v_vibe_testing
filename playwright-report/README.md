# Playwright Test Report

Open `index.html` in a browser to view the interactive test report.

## Report Features

- Filter tests by status (passed, failed, skipped)
- Filter by project or suite
- View test traces and screenshots
- Search tests by name
- Expand/collapse test details

## Running Tests

```bash
npm test                    # Run all tests
npm run test:playwright     # Run Playwright tests only
npm run test:live           # Run LIVE app tests
npm run test:waradio        # Run WARADIO app tests
npm run test:grid           # Run GRID app tests
```

Reports are generated in this directory after each test run.
